Covscan web user interface

Covscan is our internal Red Hat service that runs various static analyzers on RPM packages. Covscan tasks are automatically triggered by Errata Tool and OSCI. The tasks can also be triggered by Red Hat employees using the command-line interface provided by the covscan-client RPM package. The client can be invoked interactively as well as from CI environments and other automation.

Covscan is not Coverity. Coverity is just one of the static analysis tools that Covscan uses internally. Besides Coverity, Covscan by default uses also Cppcheck, ShellCheck, the static analyzers embedded in GCC and Clang, and the find-unicode-control tool. Other tools for static (and dynamic) analysis can be enabled on demand while submitting a Covscan task. Coverity is currently the only proprietary tool used by Covscan while all the other tools are open source. On the other hand, Coverity usually provides the most useful results from all the tools currently supported by Covscan.

Covscan is not limited to RPM packages. It can also statically analyze upstream tarballs and OpenShift containers.
