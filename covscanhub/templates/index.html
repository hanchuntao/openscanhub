Home page

Welcome to staging server of covscan hub -- web interface. This server will contain latest development version of hub and worker (program that manages coverity scans). You may expect lots of outages, so be patient, please. I might alter database structure sometimes, so as soon as your scan is finished, download all data to your computer and don't expect them to persist here.

Feel free to fill bugs or RFEs in Trac, but don't forget to mention this instance.

Required setup before usage

  1. Install latest covscan-client and it's depenencies (kobo-client, python-krbV) from here
  2. Generate a kerberos ticket (kinit)

How to use it ?

  • Check available mock configs by covscan list-mock-configs, choose the best fit for your package
  • covscan mock-build --config=... [--security/--all] [--comment=...] /path/to/srpm
  • Wait for results (email notification)
  • Download the results (.xz tarball) from WebUI
  • Extract .xz, check the results, fix the defects :), share them with upstream

For additional information please check advanced usage page
