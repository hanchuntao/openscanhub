Home page

Welcome to shiny new covscan HUB. There are several new improvements:

  • new option version-diff-build -- it scans old/new version of a package and returns the list of added/fixed defects:
    • both old/new package can be given as SRPM or by NVR of a brew build
    • example: covscan version-diff-build -p errata --base-brew-build curl-7.29.0-19.el7 --srpm ./curl-7.29.0-20.el7.src.rpm
    • file added.{err,js,html} contains a list of defects added in the new version
    • file fixed.{err,js,html} contains a list of defects fixed in the new version
  • tarballs are automatically extracted
  • open scans are not lost in the "All tasks" list
  • you may browse more filetypes (not just .logs)

Feel free to fill bugs or RFEs in Trac.

Required setup before usage

  1. Install latest covscan-client and it's depenencies (kobo-client, python-krbV) from here
  2. Generate a kerberos ticket (kinit)

How to use it?

  • Check available mock configs by covscan list-mock-configs, choose the best fit for your package
  • covscan mock-build --config=... [--security/--all] [--comment=...] /path/to/srpm
  • Wait for results (email notification)
  • Check the results on WebUI
